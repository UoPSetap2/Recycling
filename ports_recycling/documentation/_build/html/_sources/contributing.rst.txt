Contributing
============

We welcome contributions to the Portsmouth Recycling App! This section will guide you through the process.

Getting Started
---------------

Before you can contribute, you'll need to:

1. Fork the repository on GitHub.
2. Clone your fork to your local machine.
3. Set up the development environment by following the instructions in the Getting Started section of this documentation.

Making Changes
--------------

When you're ready to make a change:

1. Create a new branch for your changes.
2. Make your changes in your new branch.
3. Test your changes to make sure they work and don't break existing functionality.
4. Commit your changes and push your branch to your fork on GitHub.

Submitting a Pull Request
-------------------------

Once your changes are ready to be reviewed:

1. Go to the original repository on GitHub.
2. Click the "New pull request" button.
3. Select your fork and the branch containing your changes.
4. Click "Create pull request".

After you submit your pull request, maintainers will review your changes. They may ask for changes or clarification, so keep an eye on your pull request for comments.

Thank you for considering contributing to the Portsmouth Recycling App!
